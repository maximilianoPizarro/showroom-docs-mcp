Showroom Docs MCP Server deployed successfully!

1. Check the deployment:
   kubectl get pods -n {{ .Values.namespace }} -l app={{ include "showroom-docs-mcp.fullname" . }}

2. MCP endpoint (Streamable HTTP):
   http://{{ include "showroom-docs-mcp.fullname" . }}.{{ .Values.namespace }}.svc.cluster.local:{{ .Values.service.port }}/mcp

3. Add to OLSConfig:

   apiVersion: ols.openshift.io/v1alpha1
   kind: OLSConfig
   metadata:
     name: cluster
   spec:
     featureGates:
       - MCPServer
     mcpServers:
       - name: {{ .Values.olsConfig.mcpServerName }}
         timeout: {{ .Values.olsConfig.mcpServerTimeout }}
         url: 'http://{{ include "showroom-docs-mcp.fullname" . }}.{{ .Values.namespace }}.svc.cluster.local:{{ .Values.service.port }}/mcp'

4. Available MCP Tools:
   - searchDocs:      Search documentation by keywords
   - listDocSections: List all available document sections
   - getDocSection:   Get full content of a specific document
   - getDocSummary:   Get knowledge base overview and statistics

5. Example questions for Lightspeed:
   - "How do I install OpenShift Service Mesh?"
   - "What is the Neuralbank workshop about?"
   - "Explain Developer Hub architecture"

{{- if .Values.inspector.enabled }}

6. MCP Inspector (testing UI):

   Get the Inspector URL:
   oc get route {{ include "showroom-docs-mcp.fullname" . }}-inspector -n {{ .Values.namespace }} -o jsonpath='{.spec.host}'

   The Inspector auto-connects to the MCP server on startup.
   If it disconnects, use the auto-connect URL:

   https://<INSPECTOR_HOST>/?autoConnect=http://{{ include "showroom-docs-mcp.fullname" . }}.{{ .Values.namespace }}.svc.cluster.local:{{ .Values.service.port }}/mcp&tab=tools

7. Inspector Chat with Granite (Developer Sandbox):

   To test MCP tools with an LLM from the Inspector:
   a) Open the Inspector and click the "Chat" tab
   b) Click "Configure API Key"
   c) Configure:
      - Provider: OpenAI (compatible with vLLM/RHOAI)
      - Model:    granite-3-8b-instruct
      - API Key:  your-api-token
      - Base URL: http://<model-service>.<namespace>.svc.cluster.local/v1
   d) Start chatting - the Inspector will use MCP tools automatically
